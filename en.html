Jarhead's Meal Time (2022)

game design code art

A simple catch-the-falling-object game created using Phaser. The main goal was to create a game that is simple and easily accessible, hence a browser-based game with familiar gameplay was chosen. Created for an online event in pictsquare. During the event period, after beating the game, a code will be shown which could then be used to print a special image from any nearby convenience store in Japan with the network printing feature.

Platform: Browser
Project Length: 2 weeks
Engine and tools: Phaser, Clip Studio Paint

Escapism (2019)

game design code art

A game created for my Computer Science Bachelor's Degree thesis defense. Created in Unity for the Windows (PC) platform. The main goal is to create a game based on a pre-defined issue (in this case, mental health). A children's book illustration style was chosen to convey the atmosphere of the story. Yarn Spinner was chosen as a tool to aid with the complex dialogue system.

Platform: Windows PC
Project Length: 1 year
Engine and tools: Unity, Yarn Spinner, Easy Paint Tool SAI

Ongoing Project

code art

I have a great deal of interest in Augmented Reality (AR) and am currently experimenting with combining AR and printed media to add interactivity to an otherwise static medium.
The idea stemmed from a postcard I happened upon in a stationery shop in the streets of Kyoto. Upon scanning a QR code, it would redirect you to a website where a video will play on the postcard with the help of AR technology. This sparked the idea of adding a layer of interactivity with AR to a piece of acrylic merchandise I have created. Since creating an application for this specific purpose would add a layer of complexity not only in development but also from an end-user perspective, I opted to use Instagram's AR effect creation tool, Meta Spark Studio to simplify these issues.
