Jarhead's Meal Time (2022)

game design code art

A simple catch-the-falling-object game created using Phaser. The main goal was to create a game which is simple and easily-accessible, hence a browser-based game with a familiar gameplay was chosen. Originally created for an online event in pictsquare. During the event period, a code would be shown after the player has completed the game, which could then be used to print a special image from any nearby convenience store in Japan with the network printing feature.

Platform: Browser
Project Length: 2 weeks
Engine and tools: Phaser, Clip Studio Paint

Escapism (2019)

game design code art

A game created for my Computer Science Bachelor's Degree thesis defense. Created in Unity for Windows (PC) platform. The main goal is to create a game based on a pre-defined issue (in this case, mental health). A children's book illustration style was chosen to convey the atmosphere of the story. Since the game is very dialogue heavy, Yarn Spinner was chosen to aid with the overall dialogue system.

Platform: Windows PC
Project Length: 1 year
Engine and tools: Unity, Yarn Spinner, Easy Paint Tool SAI
